# LARP Video Scripts

These short scripts can be used to create TikTok, YouTube Shorts, or Twitter videos demonstrating LARP. They are mapped to the 8 lessons in the `larp learn` interactive tutorial.

---

## Video 1: "Saying Something in LARP" (Lesson 1)
**Length:** 15 seconds
**Visual:** Screen recording of a terminal and code editor. Fast-paced typing.
**Audio/Voiceover:**
* "Tired of `console.log` and `System.out.println` just to say hello?"
* (Typing `say "Hello, world!"` on screen)
* "In LARP, you just use the word `say`."
* (Runs code, the terminal prints `Hello, world!`)
* "It's that simple. Try it today at larplang.dev"

---

## Video 2: "Storing Values without Confusing Symbols" (Lesson 2)
**Length:** 20 seconds
**Visual:** Code editor side-by-side with the output terminal.
**Audio/Voiceover:**
* "Setting variables in programming can be confusing for beginners."
* (Typing `set name to "Pranjal"`)
* "In LARP, it's plain English. `set name to Pranjal`."
* (Typing `say "Hello, " + name`)
* "Then just say it."
* (Runs code, terminal prints `Hello, Pranjal`)
* "Readable, clean, and intuitive."

---

## Video 3: "Making Decisions in Plain English" (Lesson 3)
**Length:** 25 seconds
**Visual:** Code editor showing an if-statement.
**Audio/Voiceover:**
* "How do you write logic in LARP? Like you're writing an essay."
* (Typing `set score to 85`)
* (Typing `if score > 60: say "You passed!"`)
* "If the score is over 60, you passed."
* (Typing `otherwise: say "Try again" end`)
* "Otherwise, try again. Don't forget the `end`!"
* (Runs code)
* "It compiles directly to JavaScript, so it's crazy fast."

## Video 4: "Repeating with Loops" (Lesson 4)
**Length:** 20 seconds
**Visual:** Code editor showing a repeat loop.
**Audio/Voiceover:**
* "Need to run something multiple times? No need for `for(let i=0; i<3; i++)`."
* (Typing `repeat 3 times:`)
* (Typing `say "LARP is fun!"`)
* (Typing `end`)
* "Just say `repeat 3 times`. Boom."
* (Runs code, prints 3 times)
* "Coding shouldn't be about memorizing syntax."

## Video 5: "Creating Functions Easily" (Lesson 5)
**Length:** 30 seconds
**Visual:** Code editor showing a function definition.
**Audio/Voiceover:**
* "Functions are reusable blocks of code. LARP makes them super easy."
* (Typing `create function greet with name:`)
* (Typing `give back "Hello, " + name + "!"`)
* (Typing `end`)
* "We just created a function that gives back a greeting."
* (Typing `say greet with "Pranjal"`)
* "Then we call it with `greet with Pranjal`. Simple, right?"

## Video 6: "Working with Lists" (Lesson 6)
**Length:** 25 seconds
**Visual:** Code editor showing list creation and iteration.
**Audio/Voiceover:**
* "Need a list of items? Try this."
* (Typing `set fruits to a list containing "apple", "mango", "grape"`)
* "Now let's print them all."
* (Typing `for each fruit in fruits: say fruit end`)
* "A clean `for each` loop that reads like a sentence."
* (Runs code, prints fruits)
* "LARP takes the pain out of learning to code."

## Video 8: "Error Handling Safely" (Lesson 8)
**Length:** 30 seconds
**Visual:** Code editor showing a try-catch block.
**Audio/Voiceover:**
* "Errors happen. But LARP makes them easy to catch without crashing."
* (Typing `try:`)
* (Typing `stop and say "something broke!"`)
* "We throw an error..."
* (Typing `if something goes wrong as error:`)
* (Typing `say "Caught: " + error message end`)
* "And we catch it with `if something goes wrong`. Pure English."
* (Runs code, prints the caught error gracefully)
* "Get started at larplang.dev"
