Native AI Integration

Stop wrestling with SDKs. Use the built-in ask ai "question" keyword to seamlessly query Large Language Models directly within your logic.

Compiles to JavaScript

LARP transpiles to clean, standard JS. Deploy your applications anywhere Node.js runs without any overhead or performance penalties.

LSP & IDE Support

Enjoy modern developer experience with our standalone Language Server Protocol implementation. Get autocomplete, diagnostics, and hover support in VSCode and Neovim.

Interactive REPL

Learn LARP right in your terminal. Run larp learn for a gamified, interactive tutorial that teaches you the syntax step-by-step.

Safe & Readable

No more "Cannot read properties of undefined". With intuitive operators like use ... otherwise, your code is both safe and beautiful to read.

Built-in Standard Library

Read files, make network requests, serve web pages, and connect to databases natively. LARP comes batteries-included.

Install LARP Globally

Run one of these commands to download the pre-compiled binary and add it to your PATH.

Windows (PowerShell)
iwr -useb https://larplang.dev/install.ps1 | iex
macOS / Linux (Bash)
curl -fsSL https://larplang.dev/install.sh | bash
